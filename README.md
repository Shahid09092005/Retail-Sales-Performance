## 📌 Project Summary

This project presents a **Retail Sales Performance Analytics Dashboard** developed in **Tableau** to analyze sales trends, profitability, inventory management, and regional performance. The solution enables stakeholders to monitor key business metrics and derive actionable insights for strategic decision-making.

---

## 🎯 Objectives

* Analyze overall business performance using KPI-driven reporting.
* Identify high-performing products, stores, and categories.
* Monitor revenue, profit, and order trends.
* Optimize inventory restocking decisions.
* Evaluate pricing strategies and profitability.
